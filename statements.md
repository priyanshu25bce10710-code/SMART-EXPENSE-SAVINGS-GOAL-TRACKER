**PROBLEM STATEMENT**
People often find it difficult to track their daily spending and stay within their budget. Without a proper system, expenses get unorganized and saving goals become hard to follow. This project solves that problem by creating a simple Python-based Expense Tracker that records expenses, categorizes them, and stores all data safely, helping users manage their money more easily and stay financially aware

**SCOPE OF THE PROJECT**
The project focuses on building a simple, menu-driven Python application that helps users record, categorize, and review their daily expenses. It allows users to save all data in external storage for future reference and supports basic features like adding expenses, viewing history, searching by category, and maintaining saving goals. The project covers file handling, data organization, and user-friendly interaction, but does not include advanced analytics, online syncing, or graphical interfaces

**TARGET USERS**
1.Students: To manage daily spending, track monthly budgets, and build good financial habits.
2.Working Professionals: To monitor expenses, categorize spending, and maintain financial records for better planning.
3.Families: To record household expenses and keep track of shared spending in an organized way

**HIGH-LEVEL FEATURES**
1.Real-Time Expense Recording
2.Automatic Category-Wise Sorting
3.Monthly Budget Tracking & Alerts
4.Search & Filter for Past Transactions
5.Data Storage Using File Handling (CSV/TXT)
